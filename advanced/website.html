Minecraft
A Private Minecraft Server

What is this?

This is a private Minecraft server where the kids can play online with their friends in a closed, safe, and moderated environment. Only players we know are allowed to join at this time.

This server is different from other Minecraft servers because:

  • It is run by a parent:
    The server is maintained by a parent who knows and plays Minecraft.
  • Nobody joins without permission:
    Players must be approved to join the server, meaning every person your kid encounters is somebody they already know from school. There are no strangers.
  • There are rules:
    The admin takes a strong position against griefing and bullying in the game. If it happens, parents will be notified. If it is really bad, the player may be temporarily or permanently banned.

The goal is to provide a place where kids can learn the basic etiquette of online gaming, but in an environment that parents can trust. By playing with friends they see daily, the kids will have a better understanding of what online behavior is appropriate (and what is not).

In other words, this server provides a tangible lesson that every player in an online game is also a real person, with real emotions that are affected by what they say and do - for better or worse.

Setup

To join the server you need to purchase and install Minecraft Java Edition on a Mac or PC, and send your username to the admin. Here's how you do that:

  1. Get a minecraft.net username:
    You will need to create an account on minecraft.net and pay for a license for Minecraft: Java Edition. The license costs a one-time fee of $26.95.
  2. Download and install Minecraft Java Edition:
    You will need to install this specific version of Minecraft. Minecraft Bedrock Edition (what you find on an iPad, Xbox, Switch, and Windows 10) won't work, because it's a different game. Instead, you need to download and install Minecraft: Java Edition for your computer. It works with Windows, Mac, and Linux.
  3. Log into Minecraft:
    When you start Minecraft it will ask for a username and password. You will use the same one you just registered on minecraft.net to buy the license.
  4. Send your username to the admin:
    The admin needs to add your username to the list of allowed players. Either email it, or send it to school on a piece of paper. All the admin needs is the username (not the password), and a parent's email so he can send the final setup instructions.

Rules of the game

Each player must follow the rules of the server.

  • Respect is everything. Absolutely no bad language, trash talk, or teasing. This isn't ok in our home and it's not ok on this server. The server keeps transcripts of players' chats with each other. Players who do this will be temporarily blocked, and their parents notified.
  • No griefing. Griefing is anything intentionally done to frustrate another player. The goal is cooperative play. Minecraft can be frustrating, but it should be because of the difficulty of the game rather than another player's actions. Examples of griefing in Minecraft include:
    • Stealing items from another player's chest.
    • Destroying another player's creations without their permission.
    • Creating TNT and lava traps.
    • Setting fires around another player.
    • Killing another player's pets or livestock
    • Leaving doors open at night.
    • Excluding other players.
    • Staying awake when everybody else is sleeping.
  • Bedtime is more important than the game. The server shuts down at 8pm each night, and restarts at 8am the following morning. Each parent can set whatever limits they see fit, but our kids get 30 minutes per day during the week, and an hour on the weekend.

Some tips for players

When you begin, you can't break blocks. You should move away from that area before building your first house.

Ender chests are a way to save your special items where no other player can access your stuff. We'll keep one near the place where players enter the game.

The server shuts down at bedtime, and everything is saved automatically. When you reconnect you'll be right where you stopped.

All players must be in a bed to skip the night. If you don't have a bed, disconnect from the game for a minute when someone says "Sleep". When you reconnect to the server, it will be morning.

If you join the game and it's the middle of the night, don't panic. Disconnect, wait 7 minutes, and connect again and it'll be daytime.

Some tips for parents

It's easy for kids to get into all-Minecraft, all-the-time mode. If you want a primer on it, ask the admin.

A lot of kids use Minecraft like virtual Legos. This is Creative mode. At some point, they may want to switch to Survival mode. We run both types of games, and joining another game type means disconnecting from the current game and joining another. Unlike in single-player, you can't change the game type back and forth in the settings.

Sometimes kids really get upset over another player's actions, even to the point of tears. Often the other player thinks they're only playing. If this happens, don't wait to say something to the admin.

The server is backed up periodically. If something really bad happens, like a painstakingly created construction is destroyed by another player, it is possible to roll back to yesterday's version of the game. This means everything that was done that day is lost, but sometimes that's needed for things to be made right.

If you have any questions or concerns at all, ask.
